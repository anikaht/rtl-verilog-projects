# RTL Verilog Projects: Combinational, Sequential, and Verification Exercises

A collection of Verilog/SystemVerilog digital design projects covering combinational logic, sequential logic, and basic verification using testbenches and waveform analysis.

## Contents

### Combinational Logic
- Full Adder
- Multiplexer
- ALU

### Sequential Logic
- D Flip-Flop
- Counter
- FSM

### Verification
- Testbenches for selected modules
- Waveform-based functional verification

## Tools
- Verilog / SystemVerilog
- ModelSim / GTKWave / course simulation tools

## Skills Demonstrated
- RTL design
- Functional verification
- Testbench development
- Debugging using waveform analysis
